# Narrative Arc Packs

Community-contributed story chapters (7–14 sessions each), selected by mood and Echo Distance,
licensed CC BY 4.0.

The arc JSON format will be documented here when the narrative engine lands (Layer 2 of the
build — see the roadmap in the main README). Until then, arc proposals are welcome as
[content pack issues](../../../issues/new?template=content_pack.md).
